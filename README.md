# my diary

### 小程序主界面:
#### 日记显示区
#### 页面跳转区	
#### 日记筛选区	

### 小程序分页面:
#### 日记编辑页：日期/时间, 日记编辑区, 添加图片, 状态信息栏
#### 日记详情页
#### 统计页：个人信息板块, 时间选择器, 日记篇数统计, 日记时间点分布统计, 心情晴雨表, 图库, 地点地图分布

